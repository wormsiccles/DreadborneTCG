Dreadborne is a trading card game (TCG) initially created by indie creator [Valentine In Wonderland](www.instagram.com/valentine.in.wonderland/). It had a distinct horror theme, and takes inspiration from Yu-Gi-Oh and Chaos Galaxy.
## My Contributions
